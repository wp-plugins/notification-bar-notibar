Add a Notification Bar to your WordPress blog with the official NotiBar plugin.

== Description ==
NotiBar WordPress plugin is a simple plugin that allows you to add a notification bar on your wordpress blog, just copy and paste your notibar code from [NotiBar.com](http://www.notibar.com) straight to your WordPress admin interface.

**Requirements:** PHP5+, WordPress 2.9.x+

**Important Links:**

* [NotiBar.com](http://www.notibar.com/)


== Installation ==
1. Upload the `notification-bar-notibar` folder and all its contents to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Paste your notibar snippet from [NotiBar.com](http://www.notibar.com/) into the text area and click save!


== Frequently Asked Questions ==

= Where do I get the NotiBar code for this plugin? =

You can get your own NotiBar account from [NotiBar.com](http://www.notibar.com/) for free.
Once you have your account, create a NotiBar and simply paste your code into the plugin's text area.
